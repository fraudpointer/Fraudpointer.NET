Detailed Description

An Assessment Session is the object returned when you call API.IClient.CreateAssessmentSession(), which is the first method that you need to call to start interacting with the FraudPointer Server.

When you want to start interaction with FraudPointer Server you need to call API.IClient.CreateAssessmentSession(). This method will return to you an instance of Models.AssessmentSession. After its creation, you use this object in the following cases:

  1. When you want to add a Models.Event object to this Assessment Session.
  2. When you want to request a Fraud Assessment
  3. When you want to embed the session id in your html content, in the hidden field value that is used by FraudPointer javascript fp.js.
