Instantiates an object that properly implements IClient interface.

Start your FraudPointer API usage from here. It will give you an instance of an object that properly implements the IClient interface.

Here is an example of usage: 

            IClient client = ClientFactory.Construct("https://production.fraudpointer.com/api",
                                                     "8312453678901abcdef123456789328");

Note that the object returned uses a default timeout of 5 seconds in its communication with the Fraudpointer Server. If you want to set the timeout in a different value, use the ClientFactory.Construct(string baseUrl, string apiKey, int webRequestTimeout) method instead, giving the timeout (in milliseconds) as an input parameter.

Parameters:
baseUrlThis is the URL of the FraudPointer API Service. It has to have the value: 
https://production.fraudpointer.com/api
apiKeyThis should be the API KEY that corresponds to the domain that you integrate FraudPointer API for. The API KEY is automatically generated when you register an Account in FraudPointer Registration Web Form. Note that your Account in FraudPointer service might have more than one domains registered. You need to provide here the API KEY that corresponds to the domain that you are going to integrate this client with.
Returns:
A valid IClient instantiated object. This is a newly created object to work with the FraudPointer Service.
